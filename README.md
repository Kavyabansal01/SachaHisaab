# рџ“’ Sacha Hisaab

**Sacha Hisaab** is a simple and offline-first ledger app built for shopkeepers, freelancers, and small business owners to manage customer and supplier accounts. The app helps users keep track of transactions, outstanding balances, and payment history without requiring an internet connection.

This project was built using **React Native** and **Expo** as a practical solution for everyday bookkeeping.


